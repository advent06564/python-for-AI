# Python for AI

[![CI](https://github.com/advent06564/python-for-AI/actions/workflows/ci.yml/badge.svg)](https://github.com/advent06564/python-for-AI/actions/workflows/ci.yml)

A comprehensive lesson plan and course for learning Python for Artificial Intelligence. Covers foundational Python programming through advanced AI concepts.

## Course Structure

```
python-for-AI/
```

## Topics Covered

- Python fundamentals
- Data structures and algorithms
- NumPy, Pandas, Matplotlib
- Machine learning basics
- Deep learning introduction

## Getting Started

```bash
git clone https://github.com/advent06564/python-for-AI.git
cd python-for-AI
# Browse syllabi/ for course materials
```
